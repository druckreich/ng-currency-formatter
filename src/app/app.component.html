How to user ngCurrencyFormatterModule

How to use ngCurrencyFormatter


    <input type="text" name="value" ngCurrencyFormatter [(ngModel)]="value">
  
    model value: {{ value1 }}

How to use ngCurrencyFormatter with properties


    <input type="text" name="value" ngCurrencyFormatter [code]="'US'" [digit]="'1.4'" [(ngModel)]="value">
  
    model value: {{ value2 }}

How to use ngCurrencyValidator


    <input type="text" class="form-control" name="value" ngCurrencyFormatter ngCurrencyValidator [code]="'US'" [digit]="'1.4'" [(ngModel)]="value3" #value3Input="ngModel">
    <div class="has-danger" *ngIf="value3Input.invalid && (value3Input.dirty || value3Input.touched)">
      This is not a valid number
    </div>
{{value3Input.errors}} 
    model value: {{ value3 }}
    control valid: {{ value3Input.valid}}
    control error: {{ value3Input.errors | json}}
